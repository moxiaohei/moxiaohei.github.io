Windows下安装Ubuntu双系统

一次偶然的机会，我了解到了一个硬盘上还可以装两个系统，这使我对此产生了浓厚的兴趣，于是便自己在网上各种搜索相关的资料，自己动手在自己的windows7(64位)下装了VM虚拟机，通过在windows系统上启动VM软件来启动优麒麟。。。关键是也没怎么用，看到别人是直接在真机上运行Linux系统，我也就在好奇心的驱使下进行了我的好奇之旅！

正文

前期准备

  1. 需要准备一个启动U盘(大于2G)
  2. 在Ubuntu官网下载自己系统所匹配的安装文件
    Ubuntu 官网
  3. 下载并安装UltraISO(制作系统启动盘所需)
    UltraISO 下载
  4. 确定打算安装Ubuntu的电脑(我的电脑已安装好Win764位)

使用UltraISO制作系统启动盘

  1. 运行UltraISO软件，在本地目录中查找到自己下载好的Ubuntu镜像文件。
  2. 在UltraISO软件上方点击“启动-写入硬盘映像”。
  3. (此时U盘已经默认被选择在硬盘驱动器上)弹出的窗口上，”勾选”刻录校验;写入方式选择:USB-ZIP+;便捷启动:写入新的硬盘主引导记录(MBR)-USB-ZIP+;设置完毕后，单击写入。
  4. 单击写入之后，会出现提示，直接选择”是”就可以。之后就等待写入镜像吧，完成之后会有汉字提示的！

Ubuntu系统安装及配置

  1. 在桌面上用鼠标右键点击:我的电脑-管理-磁盘管理；选择空间大的盘符进行压缩，一般练习用的话压缩30000(30G)的空间就可以，压缩完成后即可，不用设置盘符。
  2. 将启动U盘插在电脑上，重新启动计算机，开机时按相应的快捷键进入BIOS页面(不同的电脑型号有不同的按键，具体自行进行百度)，在Boot页面LEGACY BOOT下将U盘设置为开机启动项(使用+、-可以更改启动的顺序)，按键F10保存并退出。
  3. 在选择了U盘启动后，进入Ubuntu的安装页面，在左侧选择简体中文(有其他爱好的可以选择其他语言)，点击安装Ubuntu。
  4. 出现准备安装Ubuntu，不要勾选”安装下更新”,这个需要下载，很慢的；看自己喜好可以勾选”安装这个第三方软件”。
  5. (至关重要的一步！！！)很多人在这吃过大亏，最惨的就是换电脑了，还有的抹盘就是选择了第一个；切记选择”其他选项”。点击继续
  6. 在打开的磁盘管理中我们可以看到之前划分好的”空闲”空间，大约和我们划分的差不多，30G左右；在这个空闲空间上给Ubuntu进行划分分区，选中空闲之后，点击+进行新建分区,更改新建分区容量(所分配的内存空间大小)和挂载点(就是后面提到的”/“,”/boot”等)，我是这么划分的：引导分区 “/boot”12000M(12G),系统分区 “/“ 15000M(15G),交换分区(需要在”用于”下拉菜单中选择”交换分区”)”swap”类似于Windows的虚拟内存，一般不大于物理内存就行，个人文件分区”/home”，类似于Windows的”我的文档”,自己根据自己的需求设置。重要的一点是在安装启动引导设备选择前面划分的/boot盘，点击现在安装。
  7. 会出现”您在什么地方”，选择你喜欢的就行，键盘布局选择汉语就OK。进入用户设置，自己设置用户名及密码；进入安装页面，等待更新安装。
  8. 安装完成后重新启动计算机，在window系统下进行引导。

用EasyBCD修改系统启动项

  1. 安装EasyBCD后点击”Add New Entry”-选择Linux/BSD,Type选择GRUB,Name自己随便写,笔者写的是Ubuntu作为标识,Drive:选取我们设置的/boot分区,有Linux标记,设置完成后点击“Add Entry”。
  2. 现在你可以重新启动电脑，会发现有Win7和Ubuntu两个启动下可以选择，这下就大功告成了！！！

总结

安装完双系统之后的心情是久久不能平静，瞬间感觉逼格高了很多，别傻了，这只是一个开始，会安装并不代表会用，如果你只是为了好看，我劝你还是别装了，这个很容易导致之前的Windows崩溃或者资源丢失。以后不想继续使用Ubuntu系统可以直接在Win7里面将Ubuntu的分区格式化,而不会影响Win7操作系统,这也是与从Ubuntu启动最大的好处。
